How to Use

  1. Sign in to your application
  2. Open browser developer tools (F12)
  3. In the Console, type: await Clerk.session.getToken({template: "supabase"})
  4. Copy the returned token and paste it in the JWT field above
  5. Enter a valid transaction ID
  6. Click "Generate Installments" to create commission installments for the transaction
